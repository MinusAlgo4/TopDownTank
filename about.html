About Us

Our Story

Top Down Tank is a browser-based game website created as part of a Web Technologies project. The main aim of the website is to provide a simple but enjoyable game experience directly in the browser without requiring any downloads.

Our Mission

The project was developed to demonstrate the use of HTML, CSS, and JavaScript in building an interactive website. The game includes player movement, shooting, enemy AI, collision detection, and multiple supporting pages such as Guide, Contact, Sign In, and Sign Up.

Game on

This project also helped develop skills in user interface design, page navigation, browser storage, and game logic. In the future, the game could be improved by adding more levels, better graphics, sound effects, and a more advanced enemy system.
